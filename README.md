# 🔐 Secure Authentication API

API de autenticación segura desarrollada con Django REST Framework.

## 🚀 Features

- Registro de usuarios
- Login con JWT
- Hash de contraseñas
- Roles de usuario
- Refresh tokens

## 🛠️ Tecnologías

- Python
- Django
- Django REST Framework
- PostgreSQL
- JWT
